## yuque 语雀脚本

> yueque Script
```
{
    "TOKEN": "xxxxxxxx",                                # 用户token
    "USER_AGENT": "Rion",                               # 就是一个用户名称，可以随意写，但必须要有
    "BASE_URL": "https://customspace.yuque.com/api/v2", # 语雀基础api
    "DATA_PATH": "yuque"                                # 数据储存的文件夹名称（默认当前文件夹下创建）
}
```
